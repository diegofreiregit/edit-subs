# 🎲 Subtitle Sync Editor
A simple `.srt` editor for subtitle-audio synchronization.

## 🪪 Project Overview
This project implements a simple tool used to import `.srt` files and adjust every subtitle inside the file to a specified amount.

## 📄 How to use

1. Download the script and paste it into any directory.
2. Paste the contents of the `.srt` file inside the `paste.txt` file.
3. Run the script with `python script.py` using your terminal. The adjusted `.srt` file will be inside the output folder, named `paste_shifted.srt`.

In this repo, we have the subtitles for the movie "Project Hail Mary".
